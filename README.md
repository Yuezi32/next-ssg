# next-ssg: 基于Next.js的SSR/SSG项目脚手架演示

传统的React项目开发非常快捷，但不利于SEO的缺点越来越明显。本教材详细讲述了如何基于Next.js构建SSR/SSG项目。如果你有网页静态化的开发需求，本教程将非常适合。学习一个新框架的最佳方法就是结合项目边做边学。本文将手把手带你从零开始实战SSR/SSG，通俗易懂，千万别错过！
